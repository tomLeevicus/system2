## 认证相关错误码

| 状态码 | 说明                     | 解决方案                     |
|--------|--------------------------|------------------------------|
| 40101  | Token已过期              | 调用刷新token接口或重新登录  |
| 40102  | Token无效                | 检查Token格式或重新获取      |
| 40103  | 无效的凭证或验证码        | 检查用户名/密码/验证码       |
| 40104  | 请求未携带有效Token       | 在请求头添加Authorization    |
